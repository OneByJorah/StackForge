Quick Commands
Discover all services
# JSON registry (machine-readable) curl http://YOUR_SERVER_IP:9500/api/services.json | python3 -m json.tool # Shell env vars (source this) source <(curl -s http://YOUR_SERVER_IP:9500/api/services.sh)
Use Ollama as Hermes provider
hermes config set model.base_url http://YOUR_SERVER_IP:11434 hermes config set model.provider custom:ollama hermes config set model.default ollama/llama3.2
Add this node as an MCP server
# From another Hermes agent on the tailnet: hermes mcp add j1-noc --url http://YOUR_SERVER_IP:9500/api/services.json
